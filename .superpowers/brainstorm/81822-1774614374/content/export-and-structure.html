Key Points

  • Export module uses python-docx — parses markdown AST, maps each node to Word elements (headings, tables, lists, runs)
  • Per-team Word templates — admins upload a .docx template with custom styles, logo, headers/footers. Exports inherit the template.
  • PDF optional — via LibreOffice headless in Docker (libreoffice-still package). Only if teams need it.
  • Export scope — single message, selected messages, or full conversation. User picks from a dropdown on each message or conversation menu.
  • Nginx as reverse proxy simplifies deployment — one port exposed, handles SSL termination, serves static frontend assets
  • Same Docker image for api and worker — different entrypoint (uvicorn vs celery). Reduces build time and image management.
