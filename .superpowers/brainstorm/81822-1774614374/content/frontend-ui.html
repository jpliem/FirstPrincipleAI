Frontend Structure

  • Chat view — sidebar with mode selector, doc upload, conversation list. Main area is the chat with streaming.
  • Task forms — when a mode has a form_schema, sidebar shows structured input fields instead of free chat
  • Admin: Prompt Packs — CRUD for packs, import/export JSON manifests, inline module editor (markdown)
  • Admin: Task Modes — create modes with prompt text + optional form schema builder
  • Admin: Teams — assign packs to teams, configure model, set token thresholds
  • Admin: Users — invite, roles (admin/member), team assignment
  • Admin: LLM Providers — configure API keys, endpoints, available models per provider
  • Admin: Usage & Logs — token usage per team/user, compiled prompt logs for debugging
